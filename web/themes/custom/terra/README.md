## Summary

Terra theme is the base theme for any corporative JJAA Drupal site.

## Versions

* 1.x: minimal maintenance version, compatible with any 2.x digital product

  * 1.0.x : Drupal 8.x compatible
  * 1.1.x : Drupal 9.x compatible

* `2.x`: recommended version, with maintenance

  * `2.0.x` : compatible with Drupal 9.x and Drupal 10.x. This version depends on
Terra Base 3.x

## Requirements

The module depends on Drupal core, MSD Base and its base theme, the
Bootstrap Barrio contributed theme.

## Installation

Install as usual, see https://www.drupal.org/node/2804903 for further
information.

## Configuration

This theme does not provide any particular configuration.

## Usage

Once installed, enable at the appearance settings.

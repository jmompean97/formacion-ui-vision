Soporte base para el acceso público al sitio web.

En la mayoría de sitios web basados en Drupal se diferencia entre el lado de
administración o interno y el lado público o de presentación. Este módulo
proporciona las herramientas y configuraciones comunes requeridas en la
implementación del lado público. El área interna se apoya por su parte en [Terra
Backstage](https://gitlab.juntadeandalucia.es/msd/terra_backstage).

Esta extensión debe instalarse en aquellos productos digitales Drupal que vayan
a tener una cara pública, sea o no accesible por la ciudadanía o sólo
internamente.

## Versiones

Se mantienen las siguientes versiones:

- `1.x`: versión con las siguiente subramas

  - `1.0.x` : compatible con Drupal 9.x
  - `1.1.x` : compatible con Drupal 10.x

## Requisitos

Internamente, el módulo sólo depende de Terra Base.

## Instalación

Instalar conforme al procedimiento general. Viste
https://www.drupal.org/node/1897420 para más información.

## Actualización desde Terra Base 2.x

Terra Frontstage incluye todos los elementos relacionados directamente con el
acceso público que estaban presente en Terra Base 2.x y retirado en Terra Base
3.x, versión de la que depende este módulo.

En nuevas instalaciones, Terra Frontstage provee el mismo soporte previamente
existentes en Terra Base.

En instalaciones existentes, Terra Frontstage no altera este soporte,
manteniendose la configuración presente en el producto digital.

## Configuración

El módulo no provee opciones propias o específicas de configuración.

## Uso

El módulo ofrecer cobertura para los siguientes aspectos:

- Metatags, gestión de meta etiquetas asociadas a contenidos
- Path, gestión de ruta y generación automática de alias de URL
- Easy Breadcrumb, migas de pan
- Fontawesome, iconos
- Display Suite, configuraciones avanzadas de presentación de contenido
- Layout Builder, presentación de contenido basado en diseño visual
